WhatsApp Group Chat Analysis using Machine Learning Algorithms

This project involves exporting chat data from a WhatsApp group to analyze the frequency of messages from each participant. The analysis identifies the most active members using word clouds and compares two algorithms: Decision Tree and Naive Bayesian.
